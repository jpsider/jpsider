# Update-ConsoleTitle

## SYNOPSIS
Changes the title in the Console.

## SYNTAX

```
Update-ConsoleTitle [-Title] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Update-ConsoleTitle changes the title in the Console.

## EXAMPLES

### EXAMPLE 1
```
Update-ConsoleTitle -Title "My New Title"
```
